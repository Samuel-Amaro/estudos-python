 COMO CRIAR UMA JANELA GRAFICA USANDO O QTCREATOR

 1 - DESENHE A JANELA COM O QT DESIGNER SERVE SO PARA DESENHAR NÃO EXECUTA CODIGO E NEM MOSTRA O SOFTAWARE E SO PARA DESENHAR A JANELA E E SALVAR NUM ARQUIVO .UI

2 - COM A JANELA DESENHADA POSSO CONVERTE ESSE DESIGNER PARA UM CODIGO PYTHON USANDO UM COMANDO DE COMPILAÇÃO DO PYQT5 

3 - ABRA UM TERMINAL ATE O ARQUIVO.UI E EXECUTE O SEQUINTE COMANDO:

  pyuic5 -x arquivo.ui -o nome-arquivo-codigo-gerado.py

4 - executa o codigo gerado em python para desenhar a janela com o python

  python nome-arquivo.py
